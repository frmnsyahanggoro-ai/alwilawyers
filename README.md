# Alwi Lawyers Rebuild

Prototype website company profile untuk `Alwi Lawyers`.

## File Utama

- `index.html`
- `tentang-kami.html`
- `tim-hukum.html`
- `layanan-hukum.html`
- `galeri.html`
- `hubungi-kami.html`

## Cara Menjalankan

Karena ini versi statis, file dapat dibuka langsung di browser dengan membuka `index.html`.

Jika ingin dipreview dengan local server, salah satu cara termudah:

```powershell
cd C:\Users\Firmansyah\alwi-lawyers-rebuild
python -m http.server 8000
```

Lalu buka:

`http://localhost:8000`

## Catatan

- desain sudah responsive
- konten sudah diperbarui dari versi situs lama
- aset foto masih berupa placeholder dan siap diganti saat file asli dari klien tersedia
